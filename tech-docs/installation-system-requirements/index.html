Требования

"SGroups совместим с x86_64, amd64, armhf, arm64, s390x архитектурами."

Для установки потребуется:

linux kernel >= 3.10.0

nft --version >= v0.9.3 (Topsy)

go version >= 1.19

postgresql >= 14.8
