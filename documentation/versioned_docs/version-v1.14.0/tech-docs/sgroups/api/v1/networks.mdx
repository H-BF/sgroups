<h4 className="custom-heading">Пример использования</h4>

```bash
curl '127.0.0.1:9007/v1/list/networks' \
--header 'Content-Type: application/json' \
--data '{
    "neteworkNames": ["network-example"]
}'
```

sequenceDiagram
