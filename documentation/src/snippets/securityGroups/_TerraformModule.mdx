#### [Пример использования](https://github.com/H-BF/swarm-spec-template/blob/master/spec/sg-example.yaml#L2)

```yaml
name: sg-example
cidrs:
  - 10.0.0.0/24
  - 11.0.0.0/24
default_rules:
  access:
    default:
      logs: true
      trace: true
      action: ACCEPT

    icmp:
      action: DROP
      priority: 0
      logs: true
      trace: true
      type: [0, 8]

    icmp6:
      action: DROP
      priority: 0
      logs: true
      trace: true
      type: [0, 8]
```
