<div className="text-justify">

<h4 className="custom-heading">Пример использования</h4>

```bash
curl '127.0.0.1:9007/v1/sg-sg-icmp/rules' \
--header 'Content-Type: application/json' \
--data '{
    "sgFrom": ["sg-example"],
    "sgTo": ["sg-example-2"]
}'
```
